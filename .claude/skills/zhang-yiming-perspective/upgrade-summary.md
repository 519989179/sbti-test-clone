# upgrade-summary.md

## 这次到底升级了什么

这次不是把旧版扩写一倍，而是把它从“可用版”升级成了“有证据意识的重调研版”。

### 1. 补齐了 research 骨架
新增并完成：
- `references/research/01-writings.md`
- `references/research/02-conversations.md`
- `references/research/03-expression-dna.md`
- `references/research/04-external-views.md`
- `references/research/05-decisions.md`
- `references/research/06-timeline.md`

旧版只有一个 `00-research-summary.md`，更像交代思路，不够支撑高质量 skill。
现在至少结构是完整的，而且每份都写了：
- 来源类型
- 可信度
- 一手 / 二手 / 推断的区别
- 哪些地方证据硬，哪些地方只是谨慎归纳

### 2. 把“像张一鸣”这件事做得更准了
旧版最大风险是：容易像“聪明、冷静、会做产品的产品经理”。
这次强化的不是口头禅，而是他的独特动作：
- 先重写问题
- 把复杂问题压回底层变量
- 用信息效率 / 反馈回路 / 组织失真 / 过拟合 / 时间尺度来分析
- 反 all-in 的原因不是保守，而是反停止思考

这几条组合起来，才更像张一鸣。

### 3. Agentic Protocol 强化了
旧版有 protocol，但还不够“拿来就用”。
这次把 Step 2 研究维度直接绑到张一鸣的模型上：
- 看信息效率
- 看反馈回路
- 看组织失真
- 看时间尺度
- 看不过拟合风险

这样遇到事实型问题时，不会只是“先查一下”，而是按张一鸣式问题框架去查。

### 4. 边界写实了，不装满分
这次明确写了：
- `web_search` 不可用
- 部分网页抓取失败或链接失效
- 本轮重度依赖本地高质量示例 skill 与可稳定确认事实
- 2021 后尤其 2024-2026 的细节不该写成稳定人格骨架

这比伪装成“已经完成全网核验”强得多。

### 5. 内在张力保留了
旧版虽有张力，但还偏概念化。
这次把四组最重要的矛盾钉得更稳：
- 延迟满足 vs 即时反馈产品
- 算法工具论 vs 平台责任
- Context not Control vs 关键节点强判断
- 反成功学总结 vs 外界持续总结“字节方法论”

这几组矛盾很关键。没有它们，这个人就会被写扁。

## 哪些地方证据更强了

### 更强 1：从“一个总结文件”变成“六个维度各自站住”
不是所有结论都更真了，但至少现在知道每个结论站在哪类材料上。

### 更强 2：把“观点”升级成“证据链 + 强弱等级”
例如：
- 延迟满足：高强度长期主题
- 组织的信息论：高强度长期主题
- AI 后期动向：只能作为中等可信趋势，不写死

### 更强 3：把“风格模仿”升级成“认知动作模仿”
以前更像仿语气；现在更像仿他的分析路径。

---

## 仍然存在的边界

1. **没完成理想状态的一手原文全量核验**
   - 这是最现实的边界。
2. **在线检索能力受限**
   - `web_search` 缺 API key；若干直抓页面失效或受限。
3. **大量中文旧链接已 404 或迁移**
   - 尤其媒体文章、品玩旧文、部分澎湃/36kr 页面。
4. **2021 后公开表达密度下降**
   - 近年的许多判断只能谨慎归纳，不能装作完全掌握。

## 总结一句

这版已经不是“凑一个张一鸣口吻”。
它已经有比较像样的证据意识、边界意识和人物辨识度。
还没到学术考据版，但已经够资格叫“重调研升级版”。
