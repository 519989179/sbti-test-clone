# 书稿版式统一模板

## 目标

把《一个人就是一个团队》各章的 Markdown 正文，统一输出为同一套 Word 书稿版式，避免每章各长各的。

## 当前统一规则

- 章标题：黑体、居中、加粗、较大字号
- 节标题（`##`）：黑体、加粗、与正文拉开间距
- 小节标题（`###`）：黑体、加粗、层级低于节标题
- 正文：宋体、首行缩进、统一行距
- 列表：统一项目符号与缩进层级
- 独立强调句：若 Markdown 中为单独一行 `**文本**`，按引导句/强调句处理
- 页面：A4，统一页边距

## 输出规则

- 原导出版 docx 保留，不覆盖
- 统一模板排版后的文件，命名为：`原文件名-出书排版版.docx`
- 默认双落地：workspace + 桌面书稿目录同时保存

## 通用脚本

脚本位置：
`/Users/zhaiyongqiang/.openclaw/workspace/tmp/render_book_docx.py`

### 用法

```bash
python3 /Users/zhaiyongqiang/.openclaw/workspace/tmp/render_book_docx.py \
  --md '/绝对路径/某章.md' \
  --template '/绝对路径/某章.docx' \
  --out '/绝对路径/某章-出书排版版.docx'
```

## 备注

这套模板当前属于“作者交稿版 / 书稿可读版”，已经明显优于纯导出版。
如果后续需要进一步贴近出版社成书版，还可以继续追加：分页控制、章首页留白、页眉页脚、页码、案例框样式、引文样式等。
